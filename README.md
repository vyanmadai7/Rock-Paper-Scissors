## Rock Paper Scissors  Modern Web Game

A designed, animated, theme toggle Rock–Paper–Scissors game built using HTML, CSS, and JavaScript.

## Features
## Four Difficulty Levels

Beginner :— Mostly random choices

Intermediate :— Slight prediction

Advanced :— Smarter pattern reading

Expert :— Hardest, adaptive AI

## Modern UI + Animations

Smooth hover effects

Winner pulse animations

Fade-in transitions

Responsive layout (mobile-friendly)

## Light & Dark Mode

Toggle between light and dark themes with a single button.

## In-Game Statistics

Player score

Win rate

Current streak

Total rounds played

Detailed game history (win/lose/draw)

## Interactive Choice Buttons

Rock, Paper, Scissors with emojis

Hover highlights

Winner/loser visual indicators

## Technologies Used

HTML5 :– Structure

CSS3 :– Styling, animations, responsive layout

Vanilla JavaScript :– Game logic, score tracking, AI behavior

No frameworks required — everything is fully lightweight and dependency-free.

## Project Structure
/
├── index.html           # Main UI + structure
├── styles (inline CSS)  # Modern theme-based design
└── script.js            # Game logic (choices, scoring, AI, theme)


## How It Works
Game Flow

User selects rock, paper, or scissors

AI selects based on difficulty rules

UI highlights winner/loser

Stats and history update instantly

Dynamic Stats

Win rate = wins / total rounds

Streak = consecutive wins

Game history stores each round’s result

Theme Toggle

Uses data-theme="dark" on html to switch CSS variables instantly.
